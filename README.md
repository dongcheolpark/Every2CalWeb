# Every2CalWeb
Every2Cal을 랩핑하는 웹 앱입니다.
Every2Cal은 파이썬으로 작성된 에브리타임 시간표를 ics파일로 내보내주는 프로그램입니다.

정말 좋은 프로그램이라 자주 써왔지만 파이썬 특유의 실행 방식 때문에 다수의 유저가 접근하기 힘들다고 생각해
접근성이 좋은 웹앱으로 랩핑해봤습니다.
